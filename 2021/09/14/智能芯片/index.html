  2. 硬件架构
    本小节主要介绍常用的处理器，在云端或嵌入式端广泛运用在深度学习训练、推理过程的加速。

    1. GPU
      图像处理器。
      GPU采用数量众多的计算单元和超长的流水线，善于处理图像领域的运算加速。但GPU无法单独工作，必须由CPU进行控制调用才能工作。CPU可单独作用，处理复杂的逻辑运算和不同的数据类型，但当需要大量的处理类型统一的数据时，则可调用GPU进行并行计算。深度学习的崛起受到GPU发展的推动。

    2. FPGA
      半定制化的人工智能专用芯片。FPGA适用于多指令，单数据流的分析，与GPU相反，因此常用于预测阶段，如云端。

      4. 定制化FPGA智能芯片DPU
        1. 基本概念
          DPU（Deep learning Processing Unit）深度学习处理器
    3. ASIC
      ASIC是指依产品需求不同而定制化的特殊规格集成电路，由特定使用者要求和特定电子系统的需要而设计、制造。
      1. TPU（Tensor Processing Unit）
        张量处理器
        谷歌专门为加速深层神经网络运算能力而研发的一款芯片，其实也是一款ASIC。TPU与同期的CPU和GPU相比，可以提供15-30倍的性能提升，以及30-80倍的效率（性能/瓦特）提升。
      2. NPU（（Neural network Processing Unit）
        神经网络处理器
        用电路模拟人类的神经元和突触结构，将神经网络结构直接以硬件形式体现。
